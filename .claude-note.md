# Nota interna del proyecto matriz-consultor

App separada (carpeta `matriz-consultor/`) de la presentación principal. Vercel + MongoDB Atlas (M0 gratis).
9 submódulos (las 9 materias técnicas del programa). Por materia: 3 docentes registran comprensión del
"consultor" + saber / saber-hacer / saber-ser; luego una "definición única" por materia.
API serverless: api/aportes.js, api/definicion.js. Conexión: lib/mongo.js (env MONGODB_URI / MONGODB_DB).
Upsert por (materiaId, profesor). Sin auth (acceso por link). Magenta CESDE #E6007E.
